pjXML - Pure JavaScript XML Library

There are several XML wrapper libraries available for JavaScript that use native XML objects (like XMLHttpRequest) to parse XML and provide DOM access to the underlying document. This library is designed to provide identical functionality on both the client and server sides for parsing, manipulating, and emitting XML documents.

usage

      <script src="js/pjxml.js"></script>

      . . .

      var doc = pjXML.parse('<root>Hello,&#160;XML.</root>');
